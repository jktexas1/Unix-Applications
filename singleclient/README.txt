Compile instructions:
    
	Type "make".
    
	This will create two exe's named "client" and "server".
    

Run instructions:
    
	Start up server first by typing "./server someport yes" to show 
	debug
	messages, or "./server someport no" for no debug messages.

        Start up client second by typing 
	"./client someserver someport yes/no" if no textfile, 
	or "./client someserver someport yes/no sometext.txt"
	
	if a textfile is included.
	
	On client side type the command when asked, and when asked if
 
	there is another command, type "1" for yes or "2" for no.
